# SkillSwap
SkillSwap is a free platform where people trade skills — you teach what you know, and learn what you want, with no money involved.

## How It Works
1. List a skill you can teach
2. Say what you want to learn in return
3. Get matched with someone who has what you need
4. Schedule sessions and swap skills

## Features
- Skill matching
- Session scheduling
- Reviews & ratings
- Content library by skill category
- Admin dashboard

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Node.js, Express, MongoDB, JWT
